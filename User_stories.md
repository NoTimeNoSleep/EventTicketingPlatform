# User Stories — Event Ticket Platform

## Persona 1: Eglė — Independent Event Organizer

1. As an event organizer,  
   I want to create a new event with a title, description, date, and location,  
   so that I can start selling tickets for the event.

2. As an event organizer,  
   I want to set ticket prices and available ticket quantity,  
   so that I can manage event sales.

3. As an event organizer,  
   I want to view ticket sales statistics,  
   so that I can monitor the popularity of my event.

4. As an event organizer,  
   I want to scan attendee QR codes at the event entrance,  
   so that I can quickly verify ticket validity.

---

## Persona 2: Lukas — Busy Attendee

1. As an event attendee,  
   I want to browse a list of events,  
   so that I can find events that interest me.

2. As an event attendee,  
   I want to filter events by date or category,  
   so that I can quickly find relevant events.

3. As an event attendee,  
   I want to view detailed event information,  
   so that I can decide whether I want to attend.

4. As an event attendee,  
   I want to purchase a ticket online,  
   so that I can reserve my spot at the event.

5. As an event attendee,  
   I want to receive a QR code after completing payment,  
   so that I can enter the event.

6. As an event attendee,  
   I want to receive reminders about upcoming events,  
   so that I do not forget to attend.

---

## Persona 3: Ieva — Platform Administrator

1. As a platform administrator,  
   I want to view and manage user accounts,  
   so that I can maintain platform integrity.

2. As a platform administrator,  
   I want to monitor system activity,  
   so that I can quickly detect issues or unusual behavior.

3. As a platform administrator,  
   I want to access event and ticket data,  
   so that I can resolve user issues and support platform operations.

---
 
## Persona 4: Aldona — Low-Tech Attendee
 
1. As a low-tech attendee,
   I want to browse and search the event catalog easily,
   so that I can find a specific event without getting confused.
 
2. As a low-tech attendee,
   I want to complete a ticket purchase in as few steps as possible,
   so that I do not get lost or abandon the process.
 
3. As a low-tech attendee,
   I want to receive my QR code by email with clear instructions,
   so that I know how to use it to enter the event.
 
4. As a low-tech attendee,
   I want to receive a reminder notification before the event,
   so that I do not miss the date or forget important details.

---

## Persona 5: Dylan — Corporate Event Buyer
 
1. As a corporate buyer,
   I want to purchase multiple tickets in a single transaction,
   so that I can efficiently secure spots for my entire team.
 
2. As a corporate buyer,
   I want to enter my company's billing details during checkout,
   so that I automatically receive a proper invoice for my accounting department.
 
3. As a corporate buyer,
   I want to input my colleagues' email addresses after purchase,
   so that the platform automatically sends each person their unique QR code ticket.
 
4. As a corporate buyer,
   I want to view a summary of my bulk order in my account,
   so that I can verify that all team members have received their tickets.
