# Minimum Viable Product (MVP) — Event Ticket Platform

The goal of the MVP is to enable event organizers to publish events and sell tickets, while allowing attendees to easily discover events and purchase tickets.

---

## MVP Features

### Event Discovery
- View a list of available events
- Basic event search

### Event Information
- View event description
- View event date, location, and ticket price

### Ticket Purchase
- Enter customer information
- Confirm booking details

### Payment
- Pay for a ticket using a credit or debit card

### Ticket Delivery
- Automatic QR code generation after payment
- Ticket sent to the attendee via email

### Event Check-in
- QR code scanning at the event entrance for ticket validation

---

## MVP Value

The MVP provides the core functionality needed for the platform to operate:

- Event organizers can create events and sell tickets directly.
- Attendees can discover events and purchase tickets quickly.
- QR code tickets allow fast and reliable event entry.
